M5Stack IIoT & Real-Time Telemetry Pipeline

Role

Engineering Intern

Organisation

A*STAR (ARTC)

Period

Sep 2022 – Apr 2023

Tech Stack

C++, M5Stack FIRE, InfluxDB, Grafana

The Challenge

A major hurdle in Industrial Additive Manufacturing (AM) is the lack of real-time feedback during the fabrication process. Because complex metal parts can take long periods to construct using Laser Powder Bed Fusion (LPBF), hidden flaws or excessive residual stresses can go unnoticed, resulting in wasted time, materials, and machine life.

To solve this, I was tasked with developing a system for real-time documentation and data acquisition to monitor the powder bed and machine status, ensuring errors could be detected and mitigated instantly.

Edge Computing with M5Stack IIoT

I integrated an M5Stack FIRE Industrial IoT device directly with a commercial EOS M 290 metal 3D printer. Using C++, I developed custom firmware to parse live machine states and provide remote monitoring capabilities.

  • Engineered logic to capture and interpret real-time machine signals.
  • Programmed the device to serve as an immediate visual and data-driven alert system for operators, significantly reducing response times to print errors.

End-to-End Data Telemetry Pipeline

Beyond edge-level alerts, the data needed to be logged and visualized for deeper process optimization. I built a comprehensive telemetry pipeline routing the machine data into a time-series database.

  • Deployed InfluxDB to continuously ingest high-frequency sensor data from the printer.
  • Developed interactive, live-updating dashboards in Grafana to visualize the data stream.
  • Configured the dashboard to simultaneously track and analyze 5 critical process parameters, specifically monitoring powder spreadability and chamber conditions.

Impact & Results

The integrated system provided unprecedented visibility into the LPBF process. Through rigorous validation experiments, the telemetry pipeline successfully identified micro-level powder bed surface deviations as small as ~50 µm. Furthermore, it reliably verified that chamber oxygen levels were maintained strictly below 0.1%, ensuring high-quality, defect-free metal prints.
